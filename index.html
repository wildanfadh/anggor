Install

Get up and running
in seconds.

Three steps to your new AI coding companion.

1

Install Anggor

Install globally using your preferred package manager.

$ bun install -g anggor
2

Configure your provider

Set your API key and choose your preferred LLM provider.

$ anggor provider use openai
3

Start coding

Navigate to your project and let Anggor help you.

$ cd my-project && anggor
